# AcidEngine Roadmap

## v1.0 (Current)

- [x] Field Contracts (types, ranges, regex, presets)
- [x] Container Contracts (unique, ordered, frozen)
- [x] Cross‑Field Validation (rules linking fields)
- [x] QualityGate (strict, recovery, audit, quarantine)
- [x] ErrorRecord with Explain Violation
- [x] Explain Engine (readable validation reports)
- [x] YAML Support (import/export contracts)
- [x] Pipeline Generator (skeleton from contract)
- [x] Pandas Integration (validate DataFrames directly)
- [x] CSV Loader
- [x] Smart Contract Generator (from data sample)
- [ ] Test Suite (pytest, CI via GitHub Actions)
- [ ] PyPI Package (`pip install acid-engine`)
- [ ] ARCHITECTURE.md (internal logic description)

## v1.1 (Next)

- [ ] AI Guard — LLM output validation module
- [ ] LangChain Integration (plugin)
- [ ] Contract Registry (versioning, catalog)
- [ ] Drift Detection (monitoring data changes)
- [ ] Enhanced Explain Engine (Markdown export, history)

## v1.5 (Planned)

- [ ] Polars Integration
- [ ] Full Pipeline Generation (auto‑generated code for standard operations)
- [ ] Contract Diff (compare versions)
- [ ] Soft Contracts (INFO / WARNING / ERROR levels)

## v2.0 (Future)

- [ ] Rust Runtime (performance)
- [ ] Kafka / Spark Streaming Plugins
- [ ] Marketplace of Contracts (public registry)
- [ ] Enterprise Features (SSO, RBAC, Audit Trail)
